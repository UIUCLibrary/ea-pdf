Folder

A single .mbox file becomes a single DArcMail email folder. A DArcMail email folder name is the full path of the .mbox file, with the following alterations:

  • The folder name does not have the '.mbox' extension.
  • The folder name is relative to the account directory.
  • All backslashes ('\') in a Windows path are changed to forward slashes ('/').
  • On Windows, where paths are not case-sensitive, all characters are lowercased.

Example. On a Windows file system, if the account directory is 

S:\Repository\Email\JohnDoe


an .mbox file with the path



S:\Repository\Email\JohnDoe\2001-2010\Inbox\Inbox.mbox


will have the folder name



2001-2010/inbox/inbox
