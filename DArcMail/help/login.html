Connecting to a DArcMail database

The main DArcMail program is the file DArcMail.py. If the .py extension has been associated with the Python3 interpreter (on Windows) or with the Terminal application (on Mac OS X), then you can double-click the file to start the program. Otherwise, you can start the program from a Windows command prompt window or from a Mac OS X Terminal window.

To load, delete, export, or browse data you must first connect to a DArcMail database file. The database connect screen is the first screen that appears when you start the DArcMail program. The connect screen asks you to connect to a SQLite database file. The database file may have any name, may be in any directory and may have any file extension, although a conventional extension would be ".db". If the database file does not already exist, it will be created and initialized with the DArcMail database schema.

After you have successfully connected to the DArcMail database, then the DArcMail.py program will show four tabs for loading, deleting, exporting, and browsing data.

A single database may hold only one email account. An existing SQLite DArcMail file can be used to hold data from another account only by first deleting all existing the mail data in the file (see Deleting data from the DArcMail database).
