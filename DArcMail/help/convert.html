Using DArcMailXml to Convert an Account from MBOX to XML

The "Convert MBOX Files to XML" screen asks for the following information:

  • Name.   This is the name of the email account.
  • Directory.   This is the account directory. This must already exist and must contain (directly or indirectly) one or more mbox-format files with the .mbox file extension.
  • Folder.   If "ALL MBOXES" is specified, then the program will convert all .mbox files in the account directory or in any directory that is a child of the account directory. If a single folder, for example "archive/deleted items", is specified, then the program will convert only message data from the MBOX file with the path "<account_directory>\archive\deleted items.mbox".

Each MBOX file will have a corresponding XML file. The XML produced for a MBOX file will be stored in the directory holding the MBOX file (that is, the corresponding folder directory).

Each message attachment will be stored in a XML file separate from the main XML file. Each attachment file is located in a subdirectory of the folder directory and is referenced from the main XML file. These subdirectories are used to prevent the creation of directories with thousands of files.

The maximum size of an XML file for a folder is 2GB. If the size of the XML file would exceed this limit. If a MBOX named "inbox.mbox" file would produce XML exceeding 2GB, the resulting XML files would have names like

inbox.aa.xml
inbox.ab.xml
...

CSV output

For each XML file there is a corresponding CSV file (with the extension ".csv" replacing the extension ".xml"). The CSV file contains one row of information for each message, with six columns:

  1. From
  2. To
  3. Date
  4. Subject
  5. Message ID
  6. SHA1 Hash
  7. Number of errors encountered in parsing the message
  8. The first error message encountered

CSV files acn be loaded into a spreadsheet program like Microsoft Excel or OpenOffice Calc.

Log file output

Besides the XML output files and the CSV files, the xml conversion process produces a log file, dm_xml.log.txt, in the account directory. The contents of this file are also displayed in a pop-up window when the XML conversion process has completed.
