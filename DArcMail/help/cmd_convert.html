CmdDArcMailXml Help

CmdDArcMailXml is a command-line version of DArcMailXml. It has the same options and produces the same outputs as DArcMailXml but lacks the graphical user interface.

You can run CmdDArcMailXml from a MacOS X term window or from a Windows command prompt window. Command-line options are as follows:

Long Option Short Option Kind Value Type Meaning Default
--help -h     display command options  
--account -a Required string account name None
--directory -d Required string account directory None
--folder -f Optional string folder name None
